博客网站搭建

Author: 杨佳男 Date: 2018-08-21  18:00:30 Category: 其他

之前使用博客园感觉太繁琐，并且内容编辑不是很方便，无用的菜单和广告页面上到处都是，后来发现Hexo这个博客框架，再结合Github就能实现完全属于自己的博客，申请属于自己的域名，三者合在一起就做出了现在的这个小天地，有需要的小伙伴可以自己试一下，很好玩，当属于自己的网站在互联网上被搜索到的时候，很有感觉，你会在搭建完成后不断通过手机或者电脑登录自己的域名，查看自己的网站，下面来介绍一下整个博客的搭建步骤。

注册GitHub

用于保存通过Hexo框架生成的网页静态文件。

安装Git

用于管理Hexo和执行相应的命令。

安装Node.js

Hexo通过Node.js来快速完成静态页面生成。

安装Hexo

CMD窗口npm install hexo-cli -g

选择主题

在Hexo官网寻找自己喜欢的主题并下载到本地Hexo主题目录下。

修改配置

修改主路径下的config.yml设置博客后端信息，修改themes中主题的config.yml设置博客主题信息。

申请域名

华为云、腾讯云、阿里云都有域名注册服务。

域名解析

在域名认证完成后，在域名解析中添加你的项目在GitHub中的IP地址。

个性化

Hexo有很多功能，可以根据自己的喜好添加。
